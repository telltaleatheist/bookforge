BookForge Reader

Reads web pages aloud through your local BookForge TTS engine.

Server

Leave host as 127.0.0.1 for a local BookForge — no token needed, it just works. To listen to another machine on your network, set that machine's BookForge server to listen on 0.0.0.0, then enter its IP below and the token from that machine. (Automatic discovery of servers on the network is planned for a later release.)

A local BookForge needs no token — this extension is trusted automatically. Only fill this in when connecting to BookForge on another machine; find it in that machine's config:
macOS: ~/Library/Application Support/bookforge-app/tts-api.json
Windows: %APPDATA%/bookforge-app/tts-api.json

Voice and CPU worker count live in the toolbar popup, under Engine settings — click the BookForge Reader icon to change them.
